Sheffield Supertram App 🚊

  • C#

    .NET

    LeafletJS

    React

    OpenStreetMap (OSM)

    Overpass Turbo

    • The Sheffield Supertram App is a mobile-first web app developed for a university project with two other team members, streamlining Sheffield’s tram network with features like journey planning, ticket purchasing, and live chat.

      👥 Collaborative project developed as part of a university course, with team members contributing to different areas.

      💻 Built with C#/.NET to handle backend logic and API endpoints, enabling real-time journey planning.

      🗺️ LeafletJS was used to create and manipulate custom maps, showcasing tram routes and dynamic data overlays.

      ⚛️ The project features a React frontend for a modern, interactive user experience.

      🌍 OpenStreetMap (OSM) data was integrated using Overpass Turbo to pull, format, and display location data for the journey planner.

      📈 Developed in an Agile environment with task management via Jira and a Kanban board, ensuring iterative development and regular feedback.

    Assessed university assignment – code samples and detailed project walk-through available upon request.

    HDN Sheffield 🏙️

  • Laravel

    MySQL

    FilamentPHP

    LeafletJS

    GeoJSON

    Tailwind CSS

    • HDN Sheffield is a Laravel-based resource for independent businesses and attractions in Sheffield, built to showcase the city's best pubs, restaurants, parks, and more. The project integrates FilamentPHP for easy content management and uses LeafletJS for dynamic mapping of neighbourhoods, which I created using GeoJSON data.

      🏙️ Neighbourhood boundaries created with GeoJSON, displayed on the site using LeafletJS for a visual overview of the city.

      📋 FilamentPHP provides a streamlined CMS for easy CRUD operations, allowing for rapid content management of businesses.

      🌍 Built with Laravel's ORM (Eloquent) and MySQL for database handling, with future plans to implement user search and filtering by neighbourhood or business type.

      🎨 Tailwind CSS is used for styling, ensuring a clean and responsive front-end experience.

      🚧 The project is a work-in-progress, with plans for further expansion and a larger business database before public launch.

    Private project – code samples and detailed project walk-through available upon request.
